# ucblib

Client library for the FreewheelBridge AIDL service. Provides `BikeServiceClient` for binding to the bridge, receiving real-time sensor data, and controlling workouts. Also includes `LeanSensor` for reading the Velocore's accelerometer and the UCB protocol frame/message definitions.

## Classes

### BikeServiceClient

AIDL client wrapper that handles binding, auto-heartbeat, reconnection, and death detection.

```java
BikeServiceClient client = new BikeServiceClient(context);
client.addListener(new BikeServiceClient.ListenerAdapter() {
    @Override
    public void onSensorData(SensorData data) {
        // data.power (watts), data.rpm, data.resistanceLevel, data.tilt
    }

    @Override
    public void onHeartRate(int bpm, String deviceName) {
        // BLE heart rate from connected HRM
    }

    @Override
    public void onServiceConnected() {
        client.startWorkout();  // start receiving sensor data
    }

    @Override
    public void onWorkoutStateChanged(boolean active, String reason) {
        // "started", "client_requested", "watchdog_timeout"
    }
});
client.bind();

// During workout — set resistance (1-100):
client.setResistance(15);

// Done:
client.stopWorkout();
client.unbind();
```

Key behaviors:
- **Auto-heartbeat**: sends keepalive to bridge every 5s while workout is active (prevents watchdog timeout)
- **Auto-reconnect**: re-binds to service on disconnect with 3s delay
- **Session management**: claims/releases exclusive session with the bridge

### SensorData

Parsed UCB STREAM_NTFCN (0x08) frame — 23 bytes of raw sensor data:

| Field | Type | Description |
|-------|------|-------------|
| `resistanceLevel` | int | Hardware resistance (1-25) |
| `rpm` | int | Cadence |
| `tilt` | int | Velocore pivot tilt value |
| `power` | float | Watts (computed by UCB firmware) |
| `crankRevCount` | long | Cumulative crank revolutions |
| `crankEventTime` | int | Last crank event timestamp |
| `error` | int | Error code (0 = OK) |
| `heartRate` | int | BPM from BLE HRM (0 if none) |
| `hrmDeviceName` | String | Connected HRM name |

### LeanSensor

Reads the Velocore's accelerometer at `/dev/input/event2` (~50Hz) for bike lean/tilt steering.

```java
LeanSensor lean = new LeanSensor();
lean.setSmoothingFactor(0.15f);  // EMA alpha (0.05=smooth, 0.3=responsive)
lean.setDeadzone(2.0f);          // degrees of dead zone

lean.start((x, y, z, leanDegrees) -> {
    // leanDegrees: positive = right, negative = left
    // Filtered with EMA + spike rejection
});

// Or poll:
float angle = lean.getLeanAngle();

lean.stop();
```

**Note**: The accelerometer requires `system_app` SELinux context (platform-signed apps only). Regular apps should use `SensorData.tilt` from the UCB stream instead.

### UcbDirectClient

Low-level TCP client that connects directly to the UCB serial bridge at TCP port 9999. Handles the UCB protocol init sequence, heartbeat loop, and streaming control. Used internally by the launcher when operating without FreewheelBridge.

### UcbFrame / UcbMessageIds

UCB protocol frame encoding/decoding and message ID constants.

## Distribution

ucblib is used in two ways:

1. **Gradle module dependency** — VeloLauncher includes it as `implementation(project(":ucblib"))`
2. **Standalone JAR** — consumer apps outside this monorepo (BikeArcade, FreeRide) include `ucblib.jar`

To build the standalone JAR:
```bash
./gradlew :ucblib:buildClassesJar
# Output: ucblib/build/libs/ucblib.jar
```

Copy to consumer apps:
```bash
cp ucblib/build/libs/ucblib.jar ../freeride/app/libs/
cp ucblib/build/libs/ucblib.jar ../velogames/metrics/libs/
```

## AIDL Stubs

ucblib contains hand-maintained copies of the AIDL-generated stubs (`IBikeService`, `IBikeListener`) under `io.freewheel.bridge`. These must match the stubs generated by the serialbridge module.

After modifying the `.aidl` files in serialbridge, regenerate and copy:
```bash
./gradlew :serialbridge:clean :serialbridge:compileDebugJavaWithJavac
# Then copy generated stubs from:
#   serialbridge/build/generated/aidl_source_output_dir/debug/out/io/freewheel/bridge/
# To:
#   ucblib/src/main/java/io/freewheel/bridge/
```
